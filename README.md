# Tu Corbata

## Descripción

Tu Corbata es un proyecto académico que simula el desarrollo de una tienda online especializada en la venta de corbatas.

El proyecto se realiza a partir de las necesidades de un cliente ficticio y comprende el análisis de requisitos, el diseño de la base de datos, la implementación en MySQL y la creación de un prototipo capaz de consultar los productos almacenados.

## Objetivos

- Analizar las necesidades del cliente.
- Identificar las entidades, atributos y relaciones del sistema.
- Diseñar el modelo entidad-relación.
- Transformar el diseño en un modelo relacional.
- Crear la base de datos TIENDA en MySQL.
- Relacionar las tablas mediante claves primarias y foráneas.
- Insertar datos de prueba.
- Conectar la aplicación con la base de datos.
- Consultar y mostrar las corbatas almacenadas.

## Contexto del cliente

El cliente necesita una tienda online para gestionar la venta de corbatas.

La aplicación deberá permitir almacenar:

- Usuarios registrados.
- Direcciones de envío.
- Corbatas clasificadas por talla, color, material y marca.
- Pedidos realizados por los usuarios.
- Productos incluidos en cada pedido.

## Tecnologías

- MySQL
- PHP
- HTML
- Git y GitHub
- MySQL Workbench o phpMyAdmin
- Trello

## Organización del proyecto

El desarrollo se organizará mediante un tablero de Trello, simulando un entorno de trabajo real.

Las tareas se dividirán en:

- Pendiente
- En proceso
- En revisión
- Finalizado

## Fases del proyecto

1. Recogida y análisis de requisitos.
2. Diseño del modelo entidad-relación.
3. Creación del modelo relacional.
4. Implementación de la base de datos.
5. Creación de relaciones entre tablas.
6. Inserción de datos de prueba.
7. Desarrollo de la conexión con MySQL.
8. Consulta y visualización de las corbatas.
9. Pruebas y documentación.
